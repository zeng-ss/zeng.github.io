Unity光照系统完全指南

本文最后更新于 2026-04-01T21:47:36+08:00

在 Unity 中，光照是决定场景氛围、真实感和性能的关键因素。本文将深入探讨 Unity 的光照系统，包括光照类型、阴影设置、全局光照以及性能优化策略。

光照类型

Unity 提供了四种主要的光照类型，每种都有其特定的用途和性能开销。

方向光

方向光模拟的是太阳或月亮等距离极远的光源。

  • 特点：光线平行照射，位置无限远，无衰减
  • 用途：作为场景的主光源，提供基础照明
  • 设置：通常每个场景只有一个方向光，通过 GameObject → Light → Directional Light 创建

点光源

点光源从一个点向所有方向均匀发射光线，类似灯泡。

  • 特点：全方位照明，有明显的衰减效果
  • 用途：模拟灯泡、火炬或局部补光
  • 衰减：Unity 使用二次方衰减公式来模拟真实物理光照

聚光源

聚光源从一个点发射出圆锥形的光线。

  • 特点：有方向性，有角度限制，衰减明显
  • 用途：模拟手电筒、舞台聚光灯或车头灯
  • 参数：通过 Spot Angle 控制圆锥的张开角度

面光源

面光源是一个矩形或圆形的发光面。

  • 特点：光线均匀分布，光照柔和
  • 限制：仅在烘焙或使用渐进光照贴图时有效，实时渲染中不可用
  • 用途：模拟天花板的灯带、电视屏幕发光

光照模式

Unity 中的每个光源都可以设置为以下三种模式之一：

模式 说明 适用场景
Realtime 完全实时计算，光源和阴影可动态变化 移动物体、日夜循环
Baked 完全预烘焙，运行时零开销 静态场景、移动端性能优化
Mixed 混合模式，部分实时，部分烘焙 需要动态阴影的静态物体

阴影设置

阴影是增强场景立体感的重要元素。

阴影类型

  • Hard Shadows：边缘清晰的阴影，性能开销较小
  • Soft Shadows：边缘柔和的阴影，更真实，开销较大

阴影质量优化

在 Quality Settings 中可以调整以下参数：

  • Shadow Resolution：阴影贴图分辨率
  • Shadow Distance：阴影显示距离
  • Shadow Cascade：阴影级联，用于改善远近距离的阴影质量

阴影级联

阴影级联通过将摄像机的视锥体分割成多个区域，为近处物体提供更高精度的阴影贴图。
建议配置：

  • 2 级联：适合大多数移动端
  • 4 级联：适合 PC 和主机平台的高质量阴影

全局光照

全局光照（Global Illumination, GI）模拟光线的间接反弹，实现光漫射和颜色溢出现象。

实时 GI

  • 使用 Enlighten 系统（Unity 2019 之前）或 自适应探针体积
  • 支持动态物体的间接光照
  • 对性能有一定消耗

烘焙 GI

  • 将光照信息预先计算并存储到光照贴图中
  • 运行时零性能开销
  • 适合静态场景

烘焙流程：

  1. 将物体标记为 Static → Contribute GI
  2. 在 Lighting 窗口中配置光照贴图参数
  3. 点击 Generate Lighting 开始烘焙

光照探针

光照探针用于为动态物体提供间接光照信息。

  • 在场景中放置探针组
  • 动态物体会自动采样附近探针的光照信息
  • 建议在光照变化明显的区域密集放置

反射探针

反射探针用于捕捉周围环境，为金属或光滑材质提供准确的反射效果。

类型 说明
Baked 预先烘焙的反射贴图，性能最优
Realtime 实时更新反射内容，开销较大
Custom 使用自定义的 Cubemap

性能优化建议

1. 减少实时光源数量

  • 限制每像素实时光源的数量（建议场景主光源使用方向光）
  • 次要光源使用烘焙模式

2. 优化阴影

  • 远处物体可以不投射阴影
  • 使用 Shadow Distance 限制阴影渲染距离
  • 移动端尽量使用硬阴影或无阴影

3. 合理使用 LOD

  • 结合 LOD 组件，远距离物体使用低精度阴影或关闭阴影

4. 光照贴图优化

  • 控制光照贴图分辨率（移动端建议 512 或 1024）
  • 合并静态物体的光照贴图图集

总结

Unity 的光照系统灵活而强大，合理的选择和配置可以在视觉效果和性能之间取得平衡。实践中建议：

  1. 优先考虑烘焙静态场景的光照
  2. 动态物体配合光照探针获取间接光
  3. 控制实时阴影的范围和质量
  4. 根据目标平台调整光照设置

本文基于 Unity 2022 LTS 版本编写，不同版本的部分功能可能有所差异。
